1. Start Your Backend Server

Make sure your backend server is running on port 3000:

node server.js

Or use the batch file:

restart-server.bat

2. Access the Frontend

Open your web browser and navigate to:

http://localhost:3000/frontend/index.html

Or simply double-click the index.html file in the frontend folder.

3. Login Credentials

Use your existing user credentials to login. If you don't have an account, create one through the registration process.

Note: Make sure your backend database has the required tables (pass_assignments, external_members, etc.) for the pass history to work properly.

6. Troubleshooting

If you encounter any issues:

  • Check that the backend server is running on port 3000
  • Verify your database connection
  • Check the browser console for any JavaScript errors
  • Ensure all required database tables exist

7. API Endpoints

The frontend uses these API endpoints:

POST /api/auth/login - User authentication
GET /api/pass-history - Pass assignment history
GET /api/dashboard/statistics - Dashboard statistics
POST /api/pass-history/:id/set-return-time - Set return time
